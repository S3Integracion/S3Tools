Pasos de instalación

Sigue este flujo recomendado para garantizar una instalación limpia y segura.

1. Descarga

Obtén el archivo .application de la versión que tu entorno requiere.

2. Verifica firma

Comprueba que el certificado ClickOnce sea válido antes de proceder.

3. Ejecuta instalador

Ejecuta el paquete en un equipo con permisos de administrador y conexión a S3.

4. Importa plantillas

Usa las carpetas Engines para personalizar formatos, ASIN batching o sitemaps.
